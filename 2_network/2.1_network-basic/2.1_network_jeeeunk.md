### 2.1 네트워크의 기초
#### 개념
- 노드와 링크가 서로 연결되어 있거나 연결되어 있지 않은 집합체
- 몇 개의 독립적인 장치가 적절한 영역 내에서 적당한 빠른 속도로 물리적 통신 채널을 통해 서로가 직접 통신할 수 있도록 지원해주는 데이터 통신 체계

##### `노드 Node` :
- 데이터를 송신, 수신, 작성 또는 저장할 수 있는 네트워크 내의 연결 지점
- 서버, 라우터, 스위치 등 네트워크 장치

##### `링크 Link` :
- 유선 또는 무선
